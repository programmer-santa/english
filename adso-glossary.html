A

Algorithm

A set of steps used to solve a problem or perform a task in programming.

B

Backend

The part of a software system that handles logic, database, and server-side processing.

C

Class

A template used in object-oriented programming to create objects with specific attributes and methods.

D

Database

An organized collection of data that can be accessed and managed electronically.

E

Entity

A real-world object or concept that has data stored in a database.

F

Framework

A collection of tools and libraries used to simplify and speed up software development.

G

Git

A version control system used to track changes in source code.

H

HTML

The standard language used to create and structure content on the web.
